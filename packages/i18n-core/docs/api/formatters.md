# API Reference: Formatters

## NumberFormatter

- `format(value, locale, options?)`
- `formatCurrency(value, locale, currency, options?)`
- `formatPercent(value, locale, options?)`
- `formatUnit(value, locale, unit, options?)`

## DateFormatter

- `format(date, locale, options?)`
- `formatDate(date, locale, options?)`
- `formatTime(date, locale, options?)`
- `formatRelative(date, locale, options?)`
