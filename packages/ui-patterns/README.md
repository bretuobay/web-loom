# @web-loom/ui-patterns

Composed UI patterns built on Web Loom UI Core behaviors.

## Overview

`@web-loom/ui-patterns` provides higher-level UI patterns that compose atomic behaviors from `@web-loom/ui-core` into complete interaction patterns. These patterns are framework-agnostic and can be used with React, Vue, Angular, or vanilla JavaScript.

## Features

- **Master-Detail Pattern**: Split-view interfaces with synchronized selection
- **Tabbed Interface Pattern**: Tab-based UIs with keyboard navigation
- **Sidebar Shell Pattern**: Collapsible navigation sidebars
- **Wizard Pattern**: Multi-step flows with validation
- **Modal Pattern**: Modal dialogs with stacking support
- **Toast Queue Pattern**: Notification queue management
- **Command Palette Pattern**: Keyboard-driven command interfaces with fuzzy search

## Installation

```bash
npm install @web-loom/ui-patterns
```

## Dependencies

This package requires:
- `@web-loom/ui-core`
- `@web-loom/store-core@0.0.4`
- `@web-loom/event-bus-core@0.0.2`

Optional peer dependencies:
- `@web-loom/mvvm-core` (for MVVM integration)
- `@web-loom/query-core` (for data fetching integration)
