# Design Core

`@web-loom/design-core` is a comprehensive, framework-agnostic design token system with built-in theming, CSS variable utilities, and a lightweight design system. It's perfect for building scalable, maintainable design systems across React, Vue, Angular, and vanilla JavaScript applications.

## Features

- **Design Tokens**: Single source of truth for colors, typography, spacing, shadows, and more.
- **CSS Custom Properties**: Automatic CSS variable generation from design tokens.
- **Dynamic Theming**: Runtime theme switching with light/dark mode support.
- **Framework Agnostic**: Works with any JavaScript framework.
- **TypeScript Support**: Fully typed APIs with excellent IDE autocomplete.

## Installation

```bash
npm install @web-loom/design-core
```

## Quick Start

### Import Pre-built CSS

For the quickest setup, import the pre-generated CSS files:

```javascript
// Import all token CSS variables
import '@web-loom/design-core/design-system';

// Or import specific token categories
import '@web-loom/design-core/css/colors.css';
import '@web-loom/design-core/css/spacing.css';
```

### Basic Usage with Utilities

```typescript
import { getTokenValue, generateCssVariablesString } from '@web-loom/design-core/utils';

// Get a specific token value
const primaryColor = await getTokenValue('colors.brand.primary');

// Generate and inject CSS variables
const cssVars = await generateCssVariablesString(':root');
const style = document.createElement('style');
style.textContent = cssVars;
document.head.appendChild(style);
```

## Theming

The `@design-core` utilities provide a robust way to define and switch between different themes.

### Defining a Theme

```typescript
import { createTheme } from '@web-loom/design-core/utils';

const darkTheme = createTheme('dark', {
  colors: {
    background: {
      page: '#121212',
    },
    text: {
      primary: '#E0E0E0',
    },
  },
});
```

### Applying and Switching Themes

```typescript
import { applyTheme, setTheme } from '@web-loom/design-core/utils';

// Apply the theme styles
await applyTheme(darkTheme);

// Activate the theme
setTheme('dark');
```

## Interop with UI packages

- UI Core and UI Patterns already read CSS custom properties introduced by Design Core. Theme toggles simply call `setTheme` while relying on CSS to update component palettes.
- For documentation or marketing sites, load `@web-loom/design-core/design-system` once per app and call `applyTheme` inside the provider so `setTheme` can freely switch between `[data-theme="light"]` and `[data-theme="dark"]` variants.
